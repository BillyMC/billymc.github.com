Github 部分

什么是 Github

安装 Git

以 ArchLinux 为例， $ sudo pacman -S git 。

安装并设置 SSH

$ sudo pacman -S openssh

生成一个 SSH KEY

  • $ ssh-keygen -t rsa -C "你的邮箱"
  • 第一次回车即可
  • 然后输入密码并确认密码
  • 回到 Github ，点击页面顶部 "Account Setting" >> SSH Public Keys >> Add anoter public key"
  • Title 可以随便，留空也行
  • 打开 ~/.ssh/id_rsa.pub 复制里面的内容
  • 将复制内容粘贴至 Key 里面
  • 点击 Add key 添加

测试

  • $ ssh -T git@github.com
  • 如果提示 Hi username ! You've successfully authenticated,but GitHub does not provide shell access. 即表示成功。

设置用户信息,两条命令

  • $ git config –global user.name "你的名字"
  • $ git config –global user.email "你的邮箱"

设置你的 Github token

  • 到 Github 点击 "Account Setting" >> "Account Admin"
  • 复制 API Token 下 Your API token is 后面那段号码
  • $ git config –global github.user 你的用户名
  • $ git config –global github.token 刚才复制的东西
