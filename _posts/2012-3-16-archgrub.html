一张 Arch Linux 安装盘。

启动进入安装

第一种方法

  • # grub
  • > root (hd0,6) # 主分区 / 分区所在的分区
  • > setup (hd0)
  • > quit
  • # reboot

第二种方法

  • # mount /dev/sda8 /mnt
  • # mount /dev/sda7 /mnt/boot
  • # mount -t proc /proc /mnt/proc
  • # mount -o bind /dev /mnt/dev
  • # chroot /mnt /bin/bash
  • # grub-install --no-floppy /dev/sda
  • # reboot
